## Head Tag

The `<head>` tag contains **page information**, not visible content.

It is used for:

- Page title
- Meta tags
- Linking CSS files
- SEO and browser settings

---

## Meta Tags

Meta tags provide **information about the webpage** to browsers and search engines.

They are written inside the `<head>` tag.

---

## Common Meta Tags

```html
<meta charset="UTF-8" />
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
<meta name="description" content="Page description" />
```
