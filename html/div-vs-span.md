## **Div**

- Block-level element
- Starts on a new line
- Used to group layout or large sections

### Example

```html
<div class="card">
  <h2>Title</h2>
  <p>Description</p>
</div>
```

## **Span**

- Inline element
- Does not start on a new line
- Used to style or target small text

### Example

```html
<p>Hello <span class="highlight">World</span></p>
```
